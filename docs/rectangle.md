# Rectangle settings

Use your Rectangle build separately:

- GitHub: `https://github.com/binbinsh/rectangle`

Recommended Rectangle settings for this bar:

- `Top edge margin`: `32`
- `Use absolute top edge margin (ignore display inset)`: enabled
- `Green zoom button uses Rectangle maximize`: enabled

These settings keep Rectangle-managed window layouts below the 32 px SketchyBar area on every display.

Recommended workflow:

- Build, sign, and install Rectangle from its own repository. This repo does not install it automatically.
